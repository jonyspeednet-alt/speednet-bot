# SpeedNet AI Bot

এটি SpeedNet Khulna-এর জন্য তৈরি একটি ফেসবুক মেসেঞ্জার বট। এই বটটি গ্রাহকদের বিভিন্ন প্রশ্নের উত্তর স্বয়ংক্রিয়ভাবে প্রদান করার জন্য Groq API এবং Llama 3.1 মডেল ব্যবহার করে।

## বৈশিষ্ট্য (Features)

- **স্বয়ংক্রিয় উত্তর:** গ্রাহকদের প্যাকেজ, অফিসের সময়, বিল পরিশোধ এবং সাধারণ সমস্যা সম্পর্কিত প্রশ্নের উত্তর দেয়।
- **বাংলা ভাষা:** সম্পূর্ণ বাংলায় যোগাযোগ করতে সক্ষম।
- **ফেসবুক ইন্টিগ্রেশন:** ফেসবুক পেজের মেসেঞ্জারের সাথে সংযুক্ত।
- **সহজ কনফিগারেশন:** এনভায়রনমেন্ট ভ্যারিয়েবল (`.env` ফাইল) এর মাধ্যমে সহজে কনফিগার করা যায়।

## সেটআপ এবং ইনস্টলেশন (Setup and Installation)

1.  **প্রজেক্টটি ক্লোন করুন:**
    ```bash
    git clone https://github.com/your-username/speednet-bot.git
    cd speednet-bot
    ```

2.  **ভার্চুয়াল এনভায়রনমেন্ট তৈরি এবং অ্যাক্টিভেট করুন:**
    ```bash
    python -m venv venv
    source venv/bin/activate  # Windows-এর জন্য: venv\Scripts\activate
    ```

3.  **প্রয়োজনীয় প্যাকেজ ইনস্টল করুন:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **.env ফাইল তৈরি করুন:**
    প্রজেক্টের রুট ডিরেক্টরিতে `.env` নামে একটি ফাইল তৈরি করুন এবং নিচের ভ্যারিয়েবলগুলো যোগ করুন:
    ```
    GROQ_API_KEY="আপনার গ্রোক এপিআই কী"
    VERIFY_TOKEN="আপনার তৈরি করা একটি ভেরিফিকেশন টোকেন"
    PAGE_ACCESS_TOKEN="আপনার ফেসবুক পেজের অ্যাক্সেস টোকেন"
    PORT=5000
    FLASK_DEBUG=True # ডেভেলপমেন্টের জন্য True, প্রোডাকশনের জন্য False
    FACEBOOK_API_VERSION="v19.0"
    ```

## অ্যাপ্লিকেশন চালানো (Running the Application)

**ডেভেলপমেন্ট সার্ভার:**
```bash
flask run
```

**প্রোডাকশন সার্ভার (Gunicorn ব্যবহার করে):**
```bash
gunicorn app:app
```
